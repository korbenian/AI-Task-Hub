# 🧠 ChatGPT Clone — React + Firebase + OpenRouter API

AI-чат приложение, вдохновлённое ChatGPT. Хранит историю чатов в Firebase, поддерживает смену темы и использует OpenRouter API для общения с искусственным интеллектом.

## 🚀 Стек технологий

- 🔵 React + TypeScript + Vite
- 🎨 Tailwind CSS
- 🔥 Firebase (Auth + Firestore)
- 🌐 Express.js (Backend)
- 🧠 OpenRouter API (альтернатива OpenAI)

---

## ✨ Возможности

💬 Чат с искусственным интеллектом в реальном времени
🧾 История сообщений сохраняется в Firestore
📁 Множественные чаты с уникальными ID
🌗 Переключение темы (тёмная/светлая)
🌍 Перевод интерфейса (i18n)
🔑 Аутентификация через email/password
🎯 Простая и чистая структура проекта

---

## ⚙️ Установка и запуск

```bash
# Установка зависимостей
npm install

# Запуск клиента
npm run dev

# Запуск сервера
cd server
npm install
node index.js
```
